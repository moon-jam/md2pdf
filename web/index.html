Welcome to md2pdf 🎉

A fast, local-first Markdown to PDF converter optimized for high-quality printing and pixel-perfect layouts.

✨ Core Features & How to Use

  • Live Pagination Preview: Write Markdown on the left, instantly see accurate paginated PDF results on the right. Editor scrolling syncs beautifully with the preview.
  • Local Folder Mounting: Click "Open Folder" to browse a local directory on your PC. Relative image paths (like ![img](images/pic.png)) will be resolved natively—no servers involved!
  • Rich Image Workflow:
    • Drag & Drop / Paste: Paste images directly into the editor to upload them instantly (saved locally).
    • Explorer Integration: Right-click images in the File Explorer sidebar to preview them in a lightbox, or seamlessly insert them into your text.
  • Granular Settings: Toggle page numbers, select page sizes (A4, Letter, etc.), or write Custom CSS for ultimate layout control.
  • Export: Click "Save as PDF" when you are ready. Pro tip: type <!-- pagebreak --> in your text to force a page break!
